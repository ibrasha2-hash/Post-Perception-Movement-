### **📋 Work Documentation Template**

Use this template when creating a Post-Perception artwork:

```markdown
# [Work Title]

## 1. The Question
What question does this work pose? (No answer—just the question)

## 2. Description
Technical/visual description (2-3 paragraphs)
- Medium
- Dimensions/duration
- Materials
- Interactive elements

## 3. Sensory Dimensions Tested
Which senses does it engage?
- Visual
- Auditory
- Tactile
- Multi-sensory
- Temporal

## 4. Which Principle(s) Does It Demonstrate?
(From the 9 principles)

## 5. What Is the Viewer's Role?
- Physical participant?
- Decision-maker?
- Contributor?
- Witness?

## 6. How to Experience It
Practical information:
- Duration
- Space requirements
- Necessary equipment
- Accessibility considerations

## 7. Theoretical Lineage
Which philosophers/theorists/movements relate to this work?

## 8. Perceptual Capacities Tested
What specific perceptual abilities does this test?

## 9. Evaluation Criteria Score
(Optional: self-score using the 9 criteria)

## 10. Contributing Artist
Name, contact, date created
```

### **🎨 Practical Exercises**

**Exercise 1: The Sensory Disruption**
Find a familiar object. List 5 ways to disrupt expectations about it using one of the five types of paradox (visual-linguistic, visual-auditory, visual-tactile, functional-contextual, temporal-existential).

**Exercise 2: The Perceptual Test**
Design a 5-minute experience that tests a single perceptual capacity. What would you measure?

**Exercise 3: The Open Ending**
Take a concept and create three different endings—then remove all of them. How does the work change?

## **For Researchers: Experimental Protocols**

### **Measuring Perceptual Disruption**

**Equipment:**
- Eye-tracking glasses or screen-based tracker
- ECG/heart rate monitor
- GSR (galvanic skin response) sensor
- Video camera for facial expression analysis
- Audio recorder for interviews

**Protocol:**

1. **Pre-exposure baseline** (5 min)
   - Measure baseline heart rate, GSR, eye movement

2. **Exposure** (variable, 5-30 min)
   - Track eye movement, heart rate, GSR
   - Record video for facial expression analysis

3. **Post-exposure interview** (10 min)
   - Semi-structured questions about experience
   - Self-report of disruption, cognition, emotion

4. **Follow-up** (2-7 days later)
   - Contact for memory/persistence assessment
   - Behavioral change report

## **FAQ: Frequently Asked Questions**

**Q: Can I create a Post-Perception work in any medium?**
A: Yes, but the medium must serve the perceptual disruption. See Medium Selection Strategy in the Artist's Guide.

**Q: Does my work need to have a clear message?**
A: No. Post-Perception works should pose questions, not deliver messages. Ambiguity is a strength.

**Q: How do I know if my work is "good"?**
A: Use the 9 Evaluation Criteria. Score at least 3/5 = serious consideration. 3.5+/5 = strong work.

**Q: Can I combine Post-Perception with other art movements?**
A: Yes. The movement is open to hybrid approaches. [See Contribution Guidelines →](/CONTRIBUTING.md)

**Q: What if I disagree with the framework?**
A: Excellent. Create a work that challenges the framework. Post-Perception welcomes meta-critique.
